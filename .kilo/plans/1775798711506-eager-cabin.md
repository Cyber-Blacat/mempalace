# Hybrid Room Routing Development Plan

## 项目概述
实现一个"混合房间路由"系统，结合密集（embeddings）和稀疏（BM25/keyword）信号在文档级别，然后聚合分数到房间级别。

## 目标
1. 在现有MemPalace系统基础上，实现混合检索机制
2. 提供可配置的融合算法（α * dense + β * sparse）
3. 实现文档级别→房间级别的分数聚合
4. 创建可测试的Demo模块

## 当前架构分析

### 现有系统
- `searcher.py`: 提供基于ChromaDB的密集向量搜索
- `config.py`: 配置管理系统
- `mcp_server.py`: MCP服务器，提供19个工具
- `tests/`: 完整的测试套件

### 数据模型
- 文档（drawers）存储在ChromaDB中
- 每个文档有metadata：`wing`（项目/人员）、`room`（主题）
- 当前搜索基于余弦相似度（1 - distance）

## 架构设计

### 1. 稀疏检索模块（SparseRetriever）
- 实现BM25算法
- 支持关键词匹配
- 独立于现有密集搜索

### 2. 混合融合器（HybridFuser）
- 输入：密集分数 + 稀疏分数
- 算法：`fused_score = α * dense_score + β * sparse_score`
- 可配置权重参数α和β

### 3. 房间聚合器（RoomAggregator）
- 输入：文档级别的混合分数
- 聚合策略：按房间分组，计算房间总分
- 输出：Top-K个最相关的房间

### 4. 新接口设计
- 向后兼容现有API
- 新增`hybrid_search`函数
- 支持混合模式参数

- 密集分数：ChromaDB余弦相似度（0-1）

3. 加权聚合：考虑文档在房间内的分布

## 风险与缓解

### 技术风险
1. **BM25性能问题**：对大型文档集可能较慢

class HybridRetriever:
    """结合密集和稀疏检索的混合检索器"""
    
    def __init__(self, dense_retriever, sparse_retriever, alpha=0.7, beta=0.3):
        self.dense = dense_retriever
        self.sparse = sparse_retriever
        self.alpha = alpha
        self.beta = beta
    
    def search(self, query, k=10):
        # 并行执行密集和稀疏检索
        dense_results = self.dense.search(query, k*2)
        sparse_results = self.sparse.search(query, k*2)
        
        # 融合分数
        fused_scores = self._fuse_scores(dense_results, sparse_results)
        
        # 返回Top-K结果
        return self._get_top_k(fused_scores, k)
```

class RoomAggregator:
    """文档级别分数聚合到房间级别"""
    
    def __init__(self, aggregation_method="average"):
        self.method = aggregation_method
    
    def aggregate(self, doc_scores, metadatas):
        """将文档分数按房间聚合"""
        room_scores = {}
        
        for doc_idx, score in enumerate(doc_scores):
            metadata = metadatas[doc_idx]
            room = metadata.get("room", "unknown")
            
            if room not in room_scores:
                room_scores[room] = []
            room_scores[room].append(score)
        
        # 应用聚合方法
        return self._apply_aggregation(room_scores)
```
